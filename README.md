# 🚖 OLA Data Analysis | SQL + Power BI

## 📌 Project Overview
This project analyzes ride booking data for a ride-hailing platform (OLA) to uncover insights related to booking trends, cancellations, revenue, and customer behavior.

The goal is to support data-driven decision-making for improving service efficiency and customer satisfaction.

---

## 🎯 Objectives
- Analyze booking success vs cancellations
- Identify revenue trends
- Understand ride distance patterns
- Evaluate customer and driver ratings
- Detect reasons for ride cancellations

---

## 🛠 Tech Stack
- SQL (MySQL)
- Power BI
- Excel (Dataset)

---

## 📊 Key Insights
- Around 60%+ bookings are successfully completed
- Driver cancellations are higher than customer cancellations
- Most rides fall under short-distance category
- UPI and cash are dominant payment methods
- Certain vehicle types generate higher revenue

---

## 🗄 SQL Analysis
Key queries performed:
- Retrieve successful bookings
- Average ride distance per vehicle type
- Top 5 customers by bookings
- Cancellation analysis
- Revenue calculation

---

## 📊 Power BI Dashboard
Dashboard includes:
- Ride volume over time  
- Booking status breakdown  
- Revenue by payment method  
- Cancellation reasons  
- Customer & driver ratings  

---

## ▶️ How to Use
1. Open SQL file and run queries in MySQL  
2. Load dataset in Power BI  
3. Open `.pbix` file  
4. Explore dashboard
